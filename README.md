# CNode.js 社区

> 一个令人愉快的 CNode.js 社区 PWA 版
## Notice

> 该项目不再维护，请查看[MaterialCNode](https://github.com/alexlees/MaterialCNode)项目

## 技术栈

- vue vue-router vuex
- animate.css 路由动画
- dayjs 格式化时间
- github-markdown-css
- vant 基础组件库
- Fetch 数据获取
- 基于[CNode社区API](https://cnodejs.org/api)
